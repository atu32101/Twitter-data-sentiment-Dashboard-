
Twitter Sentiment Analysis Dashboard
Project Overview
This project focuses on performing Natural Language Processing (NLP) on a large-scale dataset of over 160,000 tweets. The goal was to build a machine learning pipeline to classify sentiments and visualize the results in a professional Power BI Dashboard for real-time public opinion tracking.

 Key Features
Sentiment Classification: Categorized tweets into Positive, Negative, and Neutral using a trained Logistic Regression model.

NLP Pipeline: Implemented text cleaning, tokenization, and TF-IDF vectorization in Python.

Interactive BI Reporting: Developed a Power BI dashboard to track:

Overall Sentiment Distribution.

Volume of tweets per category.

Key keyword insights.

Data Integrity: Handled missing values and optimized data types for high-performance reporting.

 Tech Stack
Programming: Python (Pandas, Scikit-learn)

Visualization: Power BI Desktop, Plotly

Technique: Machine Learning (Supervised Learning), NLP

Logic: DAX (Data Analysis Expressions) for custom metrics

Impact
By converting raw social media text into visual intelligence, this dashboard allows stakeholders to instantly understand the "mood" of the audience, enabling data-backed strategy and faster response to public trends.
